# Footnotes, Track Your Tracks

This is small web app that provides users with their top song from each month using the Spotify API.
It is meant to visualize the changes in a person's music taste throughout the year.
Personally, when I listen back to a song that was relevant to me in a certain month I get to remember things that happened as well.
Hopefully, this can bring some good times to mind.
